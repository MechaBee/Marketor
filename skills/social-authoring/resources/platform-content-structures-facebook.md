# Platform Content Structures

Platform-specific content templates and formatting guidance for social media posts.

## Metadata Format

All content files use a markdown header (NOT YAML frontmatter) for metadata:

```markdown
# [Post Title/Hook]

## Post Metadata

**Platform**: facebook
**Format**: fb-link-post
**Topic**: travel-insurance-benefits
**Context**: erste-cc
**Revision**: 1
**Target Audience**: young-travelers
**Objective**: engagement
**CTA URL**: https://example.com/products/
**Hashtags**: #TravelInsurance, #YourBrand, #SafeTravel
**Associated Media**: image_travel-insurance_rev1.png
**Created**: 2025-10-28
**Status**: draft

---

## Post Content

[Platform-specific content structure follows...]
```

## Meta Platform (Facebook/Instagram)

### Facebook Text Post

```markdown
## Post Content

## Hook/Opening
[Attention-grabbing first line with emoji]

## Body
[2-3 short paragraphs, mobile-friendly]

## Call-to-Action
[Clear CTA with benefit]

## Engagement Prompt
[Question or prompt encouraging comments/shares]
```

**Platform Tips:**
- Use emojis strategically for visual breaks
- Keep paragraphs 2-3 lines max
- Include direct engagement prompt
- Hashtags: 3-5 relevant tags

### Facebook Link Post (AIDA Model)

```markdown
## Post Content

## Headline / Hook (Attention)
[Curiosity-driven or benefit-focused headline]

## Body Text (Interest & Desire)
[Problem + solution narrative, 3-4 short paragraphs]

[Include benefit bullets or social proof]

## Call-to-Action (Action)
[Direct CTA with link preview]

[Link to article/landing page]
```

**Length Optimization:**
- **Optimal**: 40-250 characters (maximum engagement)
- **Acceptable**: 250-500 characters (good engagement, fully visible)
- **Long-form**: 500+ characters (lower engagement, requires "See more")
- **Critical**: First 120-150 characters always visible in feed
- **Recommendation**: Keep posts under 500 characters for best performance
- **Sweet spot**: 250-400 characters balances information with visibility

**Platform Tips:**
- First line critical (appears before "See More") - optimize first 120-150 characters
- Link preview auto-generates (optimize URL metadata)
- Focus on click motivation, not full story
- AIDA structure: Attention → Interest → Desire → Action
- **Length matters**: Posts under 500 characters see higher engagement
- Use bullet points or emojis for scannable structure
- Hashtags: 3-5 focused tags (avoid clutter)

### Facebook Short Video

```markdown
## Post Content

## Video Description
[Brief description of video content, style, duration]

## Text Overlay Timeline

### 0-3 seconds
[Hook text - problem or pattern interrupt]

### 3-7 seconds
[Problem elaboration or benefit tease]

### 7-15 seconds
[Solution reveal or clear CTA]

## Caption

### Hook
[Reinforces video hook or adds context]

### Body
[Expand on video content, 2-3 lines]

### CTA
[Clear action step]

### Hashtags
[5-7 hashtags including branded tags]
```

**Platform Tips:**
- Vertical format preferred (4:5 or 9:16)
- Auto-play without sound (text overlays critical)
- First 3 seconds determine watch-through
- Duration: 10-30 seconds optimal
