# Platform Content Structures

Platform-specific content templates and formatting guidance for social media posts.

## Metadata Format

All content files use a markdown header (NOT YAML frontmatter) for metadata:

```markdown
# [Post Title/Hook]

## Post Metadata

**Platform**: facebook
**Format**: fb-link-post
**Topic**: travel-insurance-benefits
**Context**: erste-cc
**Revision**: 1
**Target Audience**: young-travelers
**Objective**: engagement
**CTA URL**: https://example.com/products/
**Hashtags**: #TravelInsurance, #YourBrand, #SafeTravel
**Created**: 2025-10-28
**Status**: draft

---

## Post Content

[Platform-specific content structure follows...]
```

---

## Cross-Platform Guidelines

### Mobile-First Design
- Short paragraphs (2-3 lines)
- Line breaks for readability
- Large, scannable text
- Visual hierarchy

### Engagement Optimization
- Strong hooks (first 3 seconds/words)
- Clear CTAs
- Audience-specific language
- Value-driven messaging

### Brand Consistency
- Maintain consistent voice
- Use branded elements
- Align with strategy guidelines
- Reference policies for tone

### Platform-Appropriate Content
- Match platform culture
- Respect format norms
- Optimize for algorithm
- Consider consumption context
