# Platform Content Structures

Platform-specific content templates and formatting guidance for social media posts.

## Metadata Format

All content files use a markdown header (NOT YAML frontmatter) for metadata:

```markdown
# [Post Title/Hook]

## Post Metadata

**Platform**: facebook
**Format**: fb-link-post
**Topic**: travel-insurance-benefits
**Context**: erste-cc
**Revision**: 1
**Target Audience**: young-travelers
**Objective**: engagement
**CTA URL**: https://example.com/products/
**Hashtags**: #TravelInsurance, #YourBrand, #SafeTravel
**Created**: 2025-10-28
**Status**: draft

---

## Post Content

[Platform-specific content structure follows...]
```

## Google Ads Platform

### Google Search Ad (Responsive Search Ad)

```markdown
## Post Content

## Headlines (10-15 options, max 30 characters each)

**Benefit-Focused (3-4):**
1. [Outcome-driven headline]
2. [Value proposition headline]
3. [Problem-solution headline]

**Keyword-Focused (3-4):**
4. [Primary keyword + benefit]
5. [Long-tail keyword variation]
6. [Location + service keyword]

**Action-Oriented (3-4):**
7. [Urgency-driven headline]
8. [CTA-focused headline]
9. [Offer/price headline]

**Social Proof (2-3):**
10. [Credibility headline]
11. [Customer count/rating]

## Descriptions (4-5 options, max 90 characters each)

1. [Problem + solution + benefit]
2. [Social proof + credibility markers]
3. [Clear CTA with urgency]
4. [Unique selling point + value]
5. [Address objection + reassurance]

## Keywords
- [Primary keyword 1]
- [Primary keyword 2]
- [Long-tail keyword 1]
- [Long-tail keyword 2]

## Negative Keywords
- [Exclude: irrelevant intent 1]
- [Exclude: low-value traffic 2]
```

**Platform Tips:**
- Google tests combinations (provide variety)
- Keyword in headline 1 boosts relevance
- Pin critical headlines to position 1
- Character limits strict (30/90)
- Focus on search intent matching
- Test 10-15 headlines, 4-5 descriptions

### Google Display Ad (Responsive Display Ad)

```markdown
## Post Content

## Short Headlines (5 options, max 30 characters)

1. [Benefit headline 1]
2. [Benefit headline 2]
3. [Question-based headline]
4. [Urgency headline]
5. [Value prop headline]

## Long Headline (1 option, max 90 characters)

[Descriptive headline combining benefit + unique feature]

## Descriptions (5 options, max 90 characters)

1. [Key benefit with emotional appeal]
2. [Social proof or credibility]
3. [Offer details with CTA]
4. [Unique value proposition]
5. [Address pain point + solution]

## Images Required
- **Landscape (1.91:1)**: 1200x628 pixels - [Description]
- **Square (1:1)**: 1200x1200 pixels - [Description]
- **Logo**: 1200x1200 (square) or 1200x300 (landscape)

## Image Descriptions
- **Landscape**: [High-quality, benefit-focused visual concept]
- **Square**: [Professional, trust-building image concept]
- **Logo**: [Clean brand logo on transparent/white background]
```

**Platform Tips:**
- Visual-first (image dominates)
- Awareness/retargeting focus
- Broad reach across Display Network
- Multiple sizes auto-generated from assets
- Keep text overlay minimal on images

**Platform Tips:**
- AI-optimized across all Google channels
- Provide maximum asset variety (15 headlines)
- Performance data trains algorithm
- Audience signals guide (not limit) targeting
- Requires robust asset library

### Google Discovery Ad

```markdown
## Post Content

## Headlines (5 options, max 40 characters)

1. [Curiosity-driven headline]
2. [Benefit-focused headline]
3. [Question-based headline]
4. [Value proposition headline]
5. [Emotional appeal headline]

## Descriptions (5 options, max 90 characters)

1. [Elaborate on benefit]
2. [Social proof element]
3. [Clear CTA with value]
4. [Unique feature highlight]
5. [Pain point address]

## Business Name
[Your Business Name - max 25 characters]

## Images Required
- **Landscape (1.91:1)**: 1200x628px - [Eye-catching, high-quality visual]
- **Square (1:1)**: 1200x1200px - [Professional, trust-building image]
- **Portrait (4:5)**: 960x1200px - [Mobile-optimized vertical visual]

## Logo
- **Square**: 1200x1200px
- **Landscape**: 1200x300px

## Call-to-Action
[Select: Learn More | Get Quote | Sign Up | Contact Us | Apply Now]
```

**Platform Tips:**
- Appears in YouTube, Gmail, Discover feeds
- Visual-first format (image quality critical)
- Discovery mindset (not search intent)
- Native ad feel (less salesy)
- Curiosity and benefit over features
