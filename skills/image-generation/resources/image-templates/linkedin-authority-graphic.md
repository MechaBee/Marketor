## Template A: Educational Framework Graphic

Best for thought leadership and save-worthy content (infographics, step-by-step guides).

### Layout Structure

1. **Header Section** (colored background)
   - Title: Bold sans-serif headline
   - Subtitle: Optional descriptive text

2. **Body Section** (light background)
   - Numbered steps or sections (3-6 items)
   - Each with: Icon, title, brief description
   - Clear visual hierarchy

3. **Footer** (optional)
   - Call-to-action or brand element

### Template Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `[framework_title]` | Main headline | "THE 5-STEP INNOVATION PROCESS" |
| `[section_count]` | Number of sections | 5 |
| `[base_color]` | Primary color (header bg) | #1F3A7F (navy blue) |
| `[accent_color]` | Icon/highlight color | #4A90E2 (light blue) |
| `[section_titles]` | List of section titles | "Research, Design, Build, Test, Launch" |

### Prompt Template

```
Create a professional infographic presenting a business framework titled
'[framework_title].'

Layout: [section_count] horizontal sections stacked vertically, each labeled
Step 1-[section_count]. Each section has:
- Sequential number icon ([section_count] in circles, [base_color] background, white number)
- Section title in bold sans-serif (dark navy, 16pt)
- Brief explanation (gray text, 11pt, max 15 words)
- Small abstract icon representing the concept

Top header: Title in white text on [base_color] background.
Body background: White with subtle [accent_color] accents.

Color scheme: [base_color] primary, [accent_color] accents, white background, dark gray text.
Clean typography (sans-serif, high contrast). The design is clean, scannable,
and save-worthy for professionals.

Aspect ratio: 1:1. High-resolution, crisp text. No people, no gradients.
```

### Example Application

```
Create a professional infographic presenting a business framework titled
'THE 5 KPIs EVERY MARKETING LEADER MUST TRACK.'

Layout: 5 horizontal sections stacked vertically, each labeled Step 1-5. Each section has:
- Sequential number icon (5 in circles, #1F3A7F background, white number)
- KPI title in bold sans-serif (dark navy, 16pt)
- Brief explanation (gray text, 11pt, max 15 words)
- Small data visualization icon (graph, gauge, chart)

Top header: Title in white text on #1F3A7F background.
Body background: White with subtle #4A90E2 accents.

Color scheme: Navy blue (#1F3A7F) primary, light blue (#4A90E2) accents, white background, dark gray text.
Clean typography (sans-serif, high contrast). The design is clean, scannable,
and save-worthy for professionals.

Aspect ratio: 1:1. High-resolution, crisp text. No people, no gradients.
```

## Template B: Authority/Social Proof Graphic

Best for testimonials, credibility building, and trust signals.

### Layout Structure

1. **Rating Element** (top)
   - Star rating or trust indicator

2. **Quote Section** (center)
   - Large, impactful testimonial text
   - Serif or bold sans-serif typography

3. **Attribution** (bottom)
   - Profile photo placeholder
   - Name and title/company

4. **Accent Elements**
   - Border or highlight line
   - Subtle background treatment

### Template Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `[testimonial_text]` | Quote content | "This program transformed my career trajectory" |
| `[client_name]` | Person's name | "Sarah Chen" |
| `[client_title]` | Title and company | "VP Marketing, TechCorp" |
| `[accent_color]` | Highlight color | #D4AF37 (gold) |
| `[background_style]` | Background treatment | "soft cream with subtle gradient" |

### Prompt Template

```
Create a professional testimonial card with a luxury aesthetic.

Layout:
- Header area: 5 gold stars aligned left
- Center: A powerful testimonial in large, bold serif type reading:
  '[testimonial_text]'
- Footer: Circular profile photo placeholder (60px), '[client_name]' in bold,
  '[client_title]' in smaller text (gray)
- Background: [background_style]
- Border: Thin [accent_color] accent line on left edge

Style: Professional, trust-inducing, suitable for B2B LinkedIn context.
The image should compel viewers to read the caption.
Typography: Large testimonial text (24-32pt serif), name (14pt bold), title (12pt regular).

Aspect ratio: 1:1. High contrast, clean design. No complex patterns.
```

## Template C: Pattern-Interrupt Graphic

Best for bold statements that stop the scroll and drive engagement.

### Layout Structure

1. **Background** (dark, high-contrast)
   - Deep navy, charcoal, or dark gray

2. **Statement** (center)
   - Large, bold text
   - Maximum impact typography

3. **Supporting Elements** (subtle)
   - Abstract shapes or visual accents
   - Reinforce concept without overwhelming

### Template Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `[bold_statement]` | Provocative headline | "YOUR BEST IDEAS ARE BEING IGNORED" |
| `[background_color]` | Dark background | #1A1A2E (dark navy) |
| `[text_color]` | Text color | #FFFFFF (white) |
| `[accent_elements]` | Visual elements | "geometric shapes, angular lines" |

### Prompt Template

```
Create an eye-catching graphic representing a contrarian business insight.

Center the graphic with a bold, provocative statement in [text_color] text on a
[background_color] background reading exactly: '[bold_statement]'

Typography: Large sans-serif font (80pt minimum), positioned center-aligned,
maximum visual impact.

Surrounding the text, add abstract visual elements ([accent_elements])
in shades of white and light gray that reinforce the concept without
overwhelming the text. The overall message must be communicated in 3 seconds.

Style: Professional yet bold, edgy aesthetic appealing to innovation-focused,
executive audience. High contrast.

Aspect ratio: 1:1. Clean, impactful, stop-scrolling aesthetic.
Text must be crisp and readable. No people.
```

### Example Application

```
Create an eye-catching graphic representing a contrarian business insight.

Center the graphic with a bold, provocative statement in white text on a
#1A1A2E (dark navy) background reading exactly: 'THE FUTURE OF WORK ISN'T DISTANT—IT'S NOW'

Typography: Large sans-serif font (80pt minimum), positioned center-aligned,
maximum visual impact.

Surrounding the text, add abstract visual elements (geometric shapes, connected nodes,
flowing angular lines) in shades of white and light gray that reinforce the concept
of digital transformation without overwhelming the text. The overall message must
be communicated in 3 seconds.

Style: Professional yet bold, edgy aesthetic appealing to innovation-focused,
executive audience. High contrast.

Aspect ratio: 1:1. Clean, impactful, stop-scrolling aesthetic.
Text must be crisp and readable. No people.
```

## Template D: Carousel Cover

Best for multi-slide series that drive 3× higher reach.

### Layout Structure

1. **Background** (bold, branded)
   - Solid color or subtle gradient

2. **Headline** (prominent)
   - Series title
   - Large, bold typography

3. **Subheading** (supporting)
   - Clarifying text

4. **Call-to-Action** (bottom)
   - "Swipe to unlock" or similar

### Template Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `[series_title]` | Main headline | "7-STEP FRAMEWORK" |
| `[series_subtitle]` | Supporting text | "TO SCALING YOUR CONSULTING PRACTICE" |


## LinkedIn Best Practices

### Text Guidelines
- Communicate core message in under 8 words for maximum impact
- Use specific numbers: "247% increase" outperforms "massive growth"
- Punchy titles (2-6 words max)

### Visual Guidelines
- Use same color palette and typography across weekly content
- 70% of LinkedIn users are on mobile—test how graphics look on phone
- First card generates curiosity; subsequent cards deliver value

### Timing & Frequency
- Post 8 AM – 2 PM weekdays for maximum reach
- Consistent 3–5 posts per week beats sporadic posting
- Longer captions (250–500 words) outperform short ones
