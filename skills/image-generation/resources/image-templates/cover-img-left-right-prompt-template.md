# A cover image template with a left and a right section

Create a wide banner/hero image with the following specifications:

## Canvas

* **Aspect Ratio:** Wide horizontal banner 3:1 (Wide panoramic banner).
* **Background Color:** `{{background_color}}`

## Layout Structure

Divide the canvas into two zones:

* **Left Zone:** \~40% width
* **Right Zone:** \~60% width

***

## LEFT ZONE: Brand Element

### Logo/Brand Text: `{{brand_name}}`

* **Font:** Thick, bold, geometric sans-serif
* **Case:** Title case (first letter capitalized)
* **Color:** `{{brand_text_color}}`
* **Size:** Large, dominant size (primary visual element)
* **Special Detail:** End the text with a period (.) that is colored `{{accent_color}}` (NOT the same as the brand text color)
* **Alignment:**&#x20;

  * Vertically: Perfect center of canvas height
  * Horizontally: Left-aligned with significant left padding (\~20% from left edge)

***

## RIGHT ZONE: Content Block

Create a vertically-centered group containing two elements:

### A. Headline Text: `{{headline_text}}`

* **Font:** Sans-serif (same family as logo), medium or regular weight
* **Weight:** Significantly thinner than the logo text
* **Size:** 30-40% of the logo text size
* **Color:** `{{headline_color}}`
* **Format:** Display as two separate lines (split text at natural break point)
* **Line Height:** Generous spacing between lines
* **Alignment:** Left-aligned (flush to invisible right-side vertical line)

### B. Call-to-Action Button: `{{cta_text}}`

* **Shape:** Pill-shaped button (fully rounded ends/semicircular caps)
* **Background Color:** `{{accent_color}}` (must match the period color in logo)
* **Text Color:** `{{cta_text_color}}`
* **Text Font:** Sans-serif, centered within button
* **Padding:** Generous horizontal padding inside pill (text should not touch edges)
* **Position:** Below headline with breathing room/margin gap
* **Alignment:** Left edge flush with headline text (same invisible vertical line)

***

## CRITICAL ALIGNMENT RULE

On the right side, create an invisible vertical line near the right edge of the canvas. The following must all align to this line:

1. First character of headline line 1
2. First character of headline line 2
3. Left edge of the pill button

This creates a clean, flush-left alignment for the entire right content block.

***

## Visual Hierarchy Summary

1. **Dominant:** Logo/Brand text with colored period (left)
2. **Secondary:** Headline text (right, top)
3. **Tertiary/Action:** CTA pill button (right, bottom)

The colored period and colored button create a visual connection across the composition.
