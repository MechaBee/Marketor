# Product Page Template

Use this template for physical products, digital products, SaaS offerings, or similar purchasable items.

## Template Guidance

**Purpose**: Drive product purchase or signup decision
**Primary CTA**: "Buy Now", "Start Free Trial", "Get Started", "Add to Cart"
**Focus**: Product benefits, features, social proof, risk reduction

## Section Definitions

### Hero Section
**Goal**: Immediately communicate product value and primary action

**Elements**:
- Headline: Product name + primary benefit (outcome-focused)
- Subheadline: Expand on benefit, address target audience
- Primary CTA: Clear action button text
- Hero image: Product in use, showing results

**Example Structure**:
```
# [Product Name]: [Primary Benefit]

[Subheadline expanding on who it's for and what problem it solves]

[Primary CTA: "Start Free Trial" | "Buy Now" | "Get Started"]

[hero-image: Product screenshot/photo showing value delivery]
```

### Benefits Section
**Goal**: Address pain points and desired outcomes

**Elements**:
- 3-5 key benefits
- Problem → Solution framing
- Specific, quantifiable when possible
- Customer-centric language (you/your)

**Framework**:
- Benefit 1: Save time/money
- Benefit 2: Reduce friction/risk
- Benefit 3: Increase results/performance

### Features Section
**Goal**: Provide detail for evaluation stage buyers

**Elements**:
- 4-6 feature descriptions
- Feature title + benefit explanation
- Technical details as needed
- "What it does" AND "why it matters"

**Format**: Feature name → Description → Benefit

### Social Proof Section
**Goal**: Build credibility and trust

**Elements**:
- 2-3 customer testimonials (with names/photos)
- Usage statistics (users, companies, success metrics)
- Trust indicators (awards, certifications, press)
- Customer logos if applicable

**Guidelines**:
- Specific testimonials (not generic)
- Include measurable results when possible
- Show diverse use cases/customers

### How It Works Section
**Goal**: Simplify adoption, reduce perceived complexity

**Elements**:
- 3-step process
- Simple, clear language
- Action-oriented steps
- Optional: visual flow diagram description

**Format**:
1. [Step 1]: Setup/Sign up
2. [Step 2]: Use/Configure
3. [Step 3]: Get results

### CTA Section
**Goal**: Final conversion push with confidence

**Elements**:
- Reinforced value proposition
- Primary CTA (repeat from hero)
- Secondary CTA (learn more, demo, contact)
- Risk reducers (guarantee, free trial, no credit card)
- Urgency/scarcity (if authentic)

**Framework**:
- Recap benefit
- Address final objection
- Clear action steps
- Remove friction

## Content Guidelines

**Tone**: Professional but accessible, benefit-focused
**Length**: Scannable - use bullets, short paragraphs
**Keywords**: Naturally integrate 3-5 target keywords
**CTAs**: Action verbs, specific outcomes, visible placement

## SEO Considerations

- Include target keyword in headline
- Use semantic variations in body
- Proper heading hierarchy (H1 → H2 → H3)
- Internal links to related products/pages
- Alt text for images (include keyword)

## Common Pitfalls to Avoid

- Feature dumping without benefit framing
- Generic testimonials lacking specificity
- Unclear or weak CTAs
- Information overload above fold
- Missing mobile optimization considerations
